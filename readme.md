##Tautology Verifier

A tautology verifier is a program that takes a **"propositional statement"** and verifies the statement
is a "tautology" or not.

Steps to run:
- Clone the code.
- Run the command
    - python tautology_verifier.py

sample input:
  - "(!A | (A & A)), (!a | (b & !a))"
 
output: 
  - **True**
  - **False**
